### Technical Implementation

- Controlled by `themeSettings.js`
- Initial load handled in `themeInit.js`
- CSS variables applied globally from `<body>` class

Key CSS variables:

--accent-color
--radius
--density
--font-scale
--neon-shadow


Everything is modular and extendable.
